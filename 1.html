HTML이란 무엇인가?

HTML(HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript).

HTML elements are the building blocks of HTML pages. With HTMLconstructs, images and other objects such as interactive forms may be embeddedinto the rendered page. HTML provides a means to create structured documents bydenoting structural semantics for text such as headings, paragraphs, lists, links,quotes and other items. HTML elements are delineated by tags, written using anglebrackets. Tags such as img and input directly introduce content into thepage. Other tags such as p surround and provide information about document textand may include other tags as sub-elements. Browsers do not display the HTMLtags, but use them to interpret the content of the page.
